# Security Policy

## Reporting Vulnerabilities

If you find a security issue in ralph-audit-commands, please report it responsibly:

1. **Do NOT open a public issue**
2. Email: dorukardahan@users.noreply.github.com
3. Include: description, steps to reproduce, impact assessment

## Scope

Ralph is a set of prompt-based audit skills for AI coding agents. Security concerns may include:
- Skills that could execute destructive operations on the host
- Prompt injection vectors in the audit output
- Information disclosure from audit reports

## Response

I'll acknowledge within 48 hours and provide a fix timeline within 7 days.
